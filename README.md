# Problem Solving Training Repository

Welcome to the **Problem Solving** repository! This project exists as a personal training ground to practice software engineering problems, algorithms, data structures, and design patterns.

## Repository Structure

The repository is organized by programming languages. Each language has its own dedicated directory containing the implementations:

- **[`python/`](./python/)**: Problem-solving scripts implemented in Python.
- **[`typescript/`](./typescript/)**: Problem-solving scripts and projects implemented in TypeScript.

## Getting Started

To get started, navigate to the language directory of your choice. Each folder contains its own `README.md` with instructions on how the files inside are organized and how to run the scripts.
